# piliwu-teach

`piliwu-teach` 是 piliwu 的國文教學專案，用來整理與開發可供課堂使用的教學工具及素材。

## 目前狀態

專案工作模式已初始化，第一個教學工具尚未建立。

## 工作方式

- 開始工作時說「開工」，讀取專案規則、Obsidian 駕駛艙並檢查 Git。
- 結束工作時說「收工」，整理進度並在確認後同步 GitHub。
- 固定規則放在 `AGENTS.md`，每日進度放在 Obsidian 專案駕駛艙。

## 安全原則

- 不提交 `.env`、API key、token、密碼或 Firebase Admin 憑證。
- 學生資料只使用座號與班級代號，不儲存真名。

## 部署

GitHub Pages 已啟用：<https://piliwu01.github.io/piliwu-teach/>

目前首頁是專案狀態頁；第一個工具完成後再替換或擴充。
